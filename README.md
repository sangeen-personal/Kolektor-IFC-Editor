# Kolektor IFC Editor

Professional IFC file analysis and property management tool.

## Features

- IFC file analysis and visualization
- Property set management
- Data export to Excel
- Data import capabilities
- 3D IFC viewer

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

## Project Structure

- `app.py` - Main application
- `core/` - Core modules for IFC processing
- `ifc_viewer/` - 3D IFC viewer component
- `assets/` - Application assets
- `static/` - Static files

## Requirements

- Python 3.8+
- ifcopenshell
- streamlit
- pandas
- openpyxl
